## Pemakaian Lokal

**Install**
```
npm install -g json-server
```

**Jalanin Server**
```
json-server --watch ./db.json --port 9876
```
> for shared via wifi, ganti host dengan ip sendiri
```
json-server --watch ~/Projects/db/db.json --host 192.168.100.5 --port 9876
```
